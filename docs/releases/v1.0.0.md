# Release v1.0.0

## Release Theme

Initial enterprise Azure landing zone reference baseline.

## Included

- Executive architecture decision pack
- Enterprise architecture document
- Azure landing zone governance standards
- Microsoft Defender security architecture
- Azure Policy catalog
- Subscription vending standard
- AKS platform standard
- Platform operating model
- Terraform modules for governance, RBAC, network, management observability, Defender, and AKS workload
- Bicep examples for governance, network, and AKS
- GitHub Actions for validation, plan, apply, container build, and Kubernetes deployment
- Kubernetes workload namespace, deployment, and HPA examples
- Portfolio DOCX playbook

## Not Included

- Real tenant IDs
- Real subscription IDs
- Real object IDs
- Production secrets
- Live Azure deployment evidence

## Intended Use

Use this release as a portfolio-grade enterprise landing zone reference and as a starting point for tenant-specific implementation planning.
